# Security Policy

Operator: **HikawaIOT**  
Contact: **factoryhack2025@gmail.com**

## Supported Scope
This repository and its automation scripts for Threads operations.

## Reporting a Vulnerability
Please report security issues by email:
- Subject: `Security Report`
- To: `factoryhack2025@gmail.com`
- Include: affected file/endpoint, reproduction steps, impact, and logs/screenshots if available.

## Response Timeline
- Initial acknowledgement: within 7 days
- Triage/update: within 14 days (best effort)
- Fix/release timing depends on severity and operational impact.

## Secrets Handling
Do not commit API keys, app secrets, or tokens to the repository.
Use environment variables or secret stores.

## Disclosure
Please allow reasonable time for remediation before public disclosure.
